Here are some projects I made. Most is the work on JavaScript used in Chrome and Opera extensions.

Chrome & Opera Extensions

QR Anything

This extension is simple one and makes use of a jQuery plugin. Simple turns the given text into QR code.

Web Travel

Its a fun extension which shows you how many miles you travelled electronically for the current page.

Awesome Tab

An exension which replaced the new tab page on Chrome with a more beautiful, informative and productive page. Opera does not currently have an API to replace the new tab so it only works in Chrome. Spent a lot of time on it and thought it was my masterpiece at some point but could not get any serious downloads.

Unblock YouTube

I got frustrated of seeing blocked videos whenever tried for something useful. So went ahead and wrote a super simple extension that just redirects all YouTube requests to playit.pk. It became the most downloaded extension on my account. It was made of only 4 lines of code. See the irony?

jQuery Plugins

Iconify

My first jQuery plugin. This plugin looks for file URLs in any element and then adds an icon related to the file.
